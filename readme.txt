Floating button for Telegram, WhatsApp, MAX and phone calls. Includes live preview and icon customization.

== Description ==

JOY Messenger Floating Button adds a customizable floating contact button to WordPress sites.

Features:
* Telegram, WhatsApp, MAX and Call buttons.
* Custom icons for the main chat button and every messenger/call button.
* Button sizes, icon scale, border radius and offsets.
* Live preview in the admin panel.
* RU / EN / ES localization files.
* Shortcode support: [messenger_button]

== Installation ==

1. Upload the plugin folder to /wp-content/plugins/ or install the ZIP from the WordPress admin panel.
2. Activate the plugin.
3. Open Joy Messenger Button in the WordPress admin menu.
4. Configure links, phone number, icons and appearance.
